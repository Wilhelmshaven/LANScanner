========================================================================
    WIN32 应用程序：LANScanner 项目概述
========================================================================

　　本程序的任务是通过ARPING的方式探知局域网内活跃的主机，并列出主机信息，同时显示本机的IP、MAC地址与网关的相应信息。
　　整个扫描过程在大约2秒内完成，网卡可以自行选择，使用多线程实现。

/////////////////////////////////////////////////////////////////////////////

　　注意：程序需要安装WinPcap才能正常运行。如缺少WinPcap，请执行目录下的安装程序。
